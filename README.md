# Manas Wakchaure — Sci-Fi Portfolio Website

A static, GitHub Pages-ready portfolio for Manas Wakchaure.

## Files

- `index.html` — main page content and SEO metadata
- `style.css` — futuristic sci-fi interface styling
- `script.js` — small interactions, tilt cards, reveal animation

## Deploy on GitHub Pages

1. Create a new repository, for example `manaswakchaure.github.io` or `portfolio`.
2. Upload `index.html`, `style.css`, and `script.js` to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/root`, then save.
6. GitHub will publish the site after the Pages build completes.

## Customization

Update the contact links, projects, and copy inside `index.html` as your public profile grows.
